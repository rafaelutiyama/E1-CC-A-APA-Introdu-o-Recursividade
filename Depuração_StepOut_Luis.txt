análise do código

o programa solicita ao usuário um tamanho de vetor, tendo que preencher com números aleatórios de 0 à 9, e após isso, ela exibe esse valores em uma ordem inversa.

step out
 
1 - o programa começa pelo método main
2- cria o vetor com tamanho N
3 - faz a chamada para exibir (antes de exibir, o preencher é chamado N - 1)
4 - início da recursão N - 1, para o método verificar se N >= 0
5 - gera um número aleatório com atribuição a v[n], o número é gerado e armazenado nele
6 - chama a recursiva para preencher (v, n - 2), ela chama a si mesmo
7 - condição de parada, termina quando N < 0, e retorna
8 -  inicia a função exibir() com o vetor preenchido
9 - início de recursão, o método verifica se N >= 0
10 - é feito a impressão do valor v[n]
11 - chamada recursiva para exibir(v, n - 2), ele se chama até N < 0
12 - condição de parada, quando N < 0, para e retorna
13 - retorno e fim da execução, todas as chamadas exibir retornam e o main encerra
